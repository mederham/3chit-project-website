How to install Raspbian on a Raspberry PI

Requirements:
  • Raspberry PI (Get it here)
  • SD card (minimum 8GB - Get it here)
  • 5V power adapter (Get it here)
  • USB type C Cable (Get it here)
  • Raspberry Pi Imager (Download here)

At first you have to download the latest version of Raspberry PI Imager.

If you work in a terminal on a Linux machine, you can just type sudo apt install rpi-imager.

After you downloaded the imager, connect your SD card to your PC and start the imager.


Now choose the OS you want to use. If not needed, we recommend using Raspbian Lite, which comes without a desktop environment.
Select the SD card you have plugged into your PC and start the process.


Otherwise you could download the image use a different tool to install the OS. (OS images, Etcher, Rufus)


After the chosen OS is installed add this file to activate SSH (Download here).

Now insert the SD-Card in the Raspberry PI and power it on. We recommend using a network cable for better internet-connection.


Now you have to find out the IPv4-Adress of your Raspberry PI so you can connect per SSH to it.

Now go to your PC and start a SSH-service. On Windows: PowerShell, on Linux/MacOS: Terminal.
Type: ssh pi@"IP-ADRESS" to start a SSH-session. And type as Passwort: raspberry


You can now enjoy Raspbian on your Raspberry PI
